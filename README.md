# CRUD-PESSOA-SPRINGBOOT
## Programa desenvolvido para estudos da tecnologia Spring boot
## Realizado um crud (Create, Retrieve, Update and Delete.) de um objeto do tipo pessoa (Id, Nome, cpf, idade e endereço)
